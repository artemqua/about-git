## Работа в Git Bash

[Установка на Windows](https://git-scm.com/download/win)


Основные команды для работы с файлами, папками и директориями:


```Bash
#examples
pwd # print working directory
cd # change directory
ls # list files in directory
mkdir  # make folder
touch table.xls # create any file
rm # remove


```
